This project analyses the relationship between climate variables and groundwater depth using machine learning techniques. Groundwater data from HydroShare and climate data from NOAA NClimGrid (2000–2022) were integrated and analysed. Several predictive models including Random Forest, XGBoost, LSTM, and Prophet were developed to forecast groundwater levels. Results show that temperature-related variables strongly influence groundwater variability, with Prophet achieving the highest prediction accuracy.

Author: Aravind Macha – MSc Data Science, University of Hertfordshire
